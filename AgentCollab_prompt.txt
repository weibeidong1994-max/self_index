4. **portable-pty 编译报错**: 确认 Xcode Command Line Tools 已安装

## 项目简介

AgentCollab 是一个本地优先的多 Agent 协作工作台（"上下文管家"），核心功能：
- 三栏布局：项目/对话列表 | 项目记忆 | 终端面板
- 内置 PTY 终端，可直接启动 Claude Code / KimiCode / Hermes 等 CLI Agent
- 自动从终端输出中蒸馏上下文（支持 LLM 增强蒸馏，默认使用 GLM API）
- 跨对话记忆聚合，按类别（决策/实现/变量/反馈/通用）组织
- 项目绑定实际文件夹，context.md 保存在项目目录的 .agenthub/ 下
- 支持同步 Kimi/Claude 的历史日志并按项目路径过滤
- 可拖拽面板宽度、项目软/硬删除

技术栈：Tauri 2 + React 19 + TypeScript + Rust (portable-pty) + SQLite + xterm.js + GLM API
