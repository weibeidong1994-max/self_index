AgentHub — 多 Agent 协作工作台

AIAgentTauriRust

项目概述

AgentHub 是一个本地优先的多 AI Agent 上下文管家，解决当你同时使用多个 AI 编程助手（如 KimiCode、Claude Code）时，它们之间互不知情、无法共享上下文的痛点。

同一项目的需求在多个 Agent 间重复解释、不同 Agent 的决策互相矛盾、项目记忆无法积累——AgentHub 通过自动记忆蒸馏和上下文注入彻底解决这些问题。

核心能力

嵌入式终端

  • 在同一窗口内启动多个 Agent 的交互式终端
  • 支持 Claude Code、KimiCode、Hermes 等主流 Agent CLI
  • 基于 xterm.js + Rust PTY Manager，原生终端体验

自动记忆蒸馏

  • 直接读取 Agent 本地日志（~/.kimi/sessions/、~/.claude/projects/）
  • LLM 五维信息提取：用户需求、Agent 回复、思考过程、涉及文件、技术决策
  • 跨会话全局聚合，避免信息碎片化

上下文注入

  • 将项目记忆以 Markdown 形式注入到 Agent 对话中
  • 新对话开始时自动加载历史上下文，无需重复解释

技术栈

React · TypeScript · Tauri · Rust · SQLite · xterm.js · LLM 蒸馏（GLM-4-flash）
