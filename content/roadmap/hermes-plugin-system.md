## 计划概述

Hermes-WebUI 的插件系统将从目前的内部扩展机制升级为开放的生态平台。第三方开发者将能够基于官方 SDK 开发自定义消息组件，并通过插件市场分发给所有 Hermes 用户。

## 主要特性

插件 SDK 将提供完整的类型定义、渲染生命周期和状态管理 API。官方市场会内置审核与评分机制，确保插件质量。对于企业用户，私有插件仓库功能将满足内部组件的安全分发需求。同时，我们计划推出"一键导入"功能，支持将 LangChain 工具、MCP Server 等外部能力快速转化为 Hermes 插件。

## 时间规划

技术方案设计已完成，预计 2026 年 Q3 启动开发，Q4 发布 Beta 版本。欢迎社区开发者提前关注并参与 SDK 早期测试。
